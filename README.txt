# Estude Comigo
Plataforma web desenvolvida para auxiliar estudantes na preparação para o ENEM, reunindo conteúdos das principais disciplinas cobradas na prova. O projeto tem como objetivo organizar os temas de maior incidência, facilitando o aprendizado de forma prática e acessível.

## Funcionalidades 
- Organização de conteúdos por disciplinas.
- Destaque para temas com maior peso no Enem.
- Iterface simples e de fácil navegação.
- Acesso rápido aos principais tópicos de estudo.

## Disciplinas abordadas

- Biologia  
- Física  
- Literatura  
- Português  
- Matemática  
- Química  
- Geografia  
- Sociologia  
- História do Brasil  

## Tecnologias utilizadas

- HTML  
- CSS  
- JavaScript
